# Quiz do Flamengo

Site estático HTML/CSS/JS com quiz temático do Flamengo.

## Estrutura do Projeto

- `index.html` - Página principal (tela de loading)
- `quiz/` - Página do quiz
- `start-tela/` - Tela inicial
- `parabens/` - Tela de vitória
- `perdeu/` - Tela de derrota
- `loading1/`, `loading2/` - Telas de carregamento
- `nf1/` - Página adicional

## Deploy no Railway

1. Faça fork ou clone este repositório
2. Conecte seu repositório GitHub ao Railway
3. O Railway detectará automaticamente o `package.json`
4. O site será servido na porta configurada automaticamente

## Execução Local

```bash
npm install
npm start
```

O site estará disponível em `http://localhost:8080`

## Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla
- http-server (para servir arquivos estáticos)
